- 📊 Superstore Sales Analysis (Excel)**
Analyzed 9,994 orders (2014–2017) across 4 regions and 3 product categories using Pivot Tables and Advanced Excel. Found that Technology drove the highest profit despite Furniture having comparable sales, and that Tables was the only sub-category running a net loss — largely tied to high discounting. Built dashboards to visualize regional and category-wise performance.

- 📉 Sales & Profit Trends Dashboard (Power BI)**
Built an interactive Power BI dashboard analyzing 8,314 transactions (2021–2024) using Data Modeling and DAX measures. Tracked sales and profit trends by category, state, and year with line charts and slicers for dynamic filtering — revealing that Technology consistently led in both sales and profit, while California and New York were the top-performing states. Used DAX to calculate YoY growth and category-wise profit margins.

- 🗄️ Employee Performance & Salary Analysis (SQL + Power BI)**
Loaded a 100-employee dataset into MySQL and used SQL queries (Joins, Aggregations, Group By) to analyze salary and performance trends. Connected Power BI directly to the MySQL database for live visualization. Found that IT had the highest average salary, while salary didn't scale linearly with performance rating — employees rated "Good" earned more on average than those rated "Excellent."
